🎬 연출 작업실 모바일 UI 전면 개편 및 서브패널 접기 기능
▶
연출 작업실 모바일 UI 전면 개편
  • 대본 작업실과 동일한 하단 슬라이딩 패널 구조로 통일
  • 하단 탭: 작품 / 연출 / 보드
    • 작품: 작품 목록 + 대본 본문 미리보기
    • 연출: 작품별 연출노트 목록 + 메모 작성
    • 보드: 스토리보드 생성 및 편집
  • 키보드 감지 시 하단 패널 자동 수축 (타이핑 방해 없음)
연출 작업실 데스크톱 개선
  • 메인 사이드바 토글 버튼을 헤더로 이동 — 메뉴 항목 가림 해소
  • 작품 목록 · 연출노트 목록 · 스토리보드 목록 패널 개별 접기(◀/▶) 지원
  • 패널 너비 전환 시 부드러운 애니메이션
스토리보드
  • 모바일에서 가로형 뷰 지원
  • 출력 기본값 변경 — 머리말/꼬리말·배경그래픽 기본 OFF
버그 수정
  • 게스트(둘러보기) 모드 검은 화면 오류 수정

🎬 씬번호 인식 형식 확장
▶
씬번호 인식 패턴 유연화

오류제출 탭 개선

  • 인식 안 되는 형식 제보하기 버튼

  • 낮/밤 칸에 입력된 시간대 텍스트 그대로 표시 (낮·밤·D·N·아침 등)

트리트먼트 개편

  • 미니시리즈 미생성 회차 자동생성 선택지 추가
타임라인 눈금 수정

  • 대본 분량에 따라 타임라인 총 분이 자동 반영 (35매→70분, 40매→80분)
작품 생성 개편
  • 단막 / 미니시리즈 / 영화 / 기타 4가지 형식 선택
  • 유형별 클라이막스 구간 자동 설정
  • 미니시리즈: 부작 수 입력 → 회차 자동 생성
구조 페이지 컬러바 개선
  • 씬 위치를 타임라인 기준 시간 비율로 계산
